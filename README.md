# Random P/W Generator

## Description

The program allows the user to enter the name of the service they
wish to create a password for along with a small description

The program saves the user's input and generates a random password according to the
length provided by the user.

the name, description and the new password are then store in a .txt file under the
same name as the service.

## How to use?

- Compile the c file using the command `gcc pwgen.c -o run`
- Execute the file `clear && ./run`
- Output file will be saved in the same directory when successful
